Time Series Model Generation and ACVF/ACF Computation Vignette

source("ts_functions.R")

This vignette serves as an example on how to use the functions within the sta380 package for simulating autoregressive (AR), moving-average (MA), and autoregressive moving-average (ARMA) models, compute their corresponding theoretical and sample autocovariance (ACVF) and autocorrelation (ACF) function, plot their data, and related functions:

  • Model Generation sta380::gen_arma(),

  • Converting ARMA models into infinite order MA models sta380::arma_to_ma(),

  • Determining causality of a model sta380::is_causal(),

  • Computing theoretical ACVF and ACF of a model sta380::get_theoretical_acvf() and sta380::get_theoretical_acf() respectively.

  • Computing sample ACVF and ACF of a model sta380::get_sample_acvf() and sta380::get_sample_acf() respectively.

The following examples will be sampling time series data from the following ARMA(1, 2), AR(1), and MA(2) models: \[X_t = 0.5X_{t - 1} + w_t + 0.5w_{t - 1} + 0.25w_{t - 2},\] \[X_t = 0.5X_{t - 1} + w_t, \text{ and}\] \[X_t = w_t + 0.5w_{t - 1} + 0.25w_{t - 2},\]

where \(w_t \overset{\text{i.i.d.}}{\sim} N(\mu = 0, \sigma_W^2 = 1).\)

Note, while we will generate data and compare results for all three models, we omit examples of utilizing the helper functions sta380::arma_to_ma() and sta380::is_causal() for AR and MA models as they are similarly utilized.

Generating Time Series Data for all models and plotting them

set.seed(1)
n <- 50
phi <- 0.5
theta <- c(0.5, 0.25)
wn <- rnorm(n)

ar <- gen_arma(n, wn, ar_coefs = phi, ma_coefs = NULL)
ma <- gen_arma(n, wn, ar_coefs = NULL, ma_coefs = theta)
arma <- gen_arma(n, wn, ar_coefs = phi, ma_coefs = theta)

plot.ts(ar$data, main = "AR(1) Simulated Data", ylab = "X_t", xlab = "t")

plot.ts(ma$data, main = "MA(2) Simulated Data", ylab = "X_t", xlab = "t")

plot.ts(arma$data, main = "ARMA(1,2) Simulated Data", ylab = "X_t", xlab = "t")

Plotting Theoretical ACVF and ACF

Here we visualize the sample and theoretical ACF and ACVF for each model.

# function to compare plots
plot_compare <- function(theoretical, sample, main, ylab) {
  lags <- 0:(length(theoretical) - 1)
  plot(lags, theoretical, type = "b", col = "blue", pch = 19,
       main = main, xlab = "Lag", ylab = ylab,
       ylim = range(c(theoretical, sample), na.rm = TRUE))
  lines(lags, sample, type = "b", col = "red", pch = 17)
  abline(h = 0)
  legend("topright", legend = c("Theoretical", "Sample"),
         col = c("blue", "red"), pch = c(19, 17), lty = 1, bty = "n")
}

# ARMA(1,2) plots
plot_compare(arma_t_acvf, arma_s_acvf,
             main = "ARMA(1,2) ACVF: Theoretical vs Sample",
             ylab = "ACVF")


plot_compare(arma_t_acf, arma_s_acf,
             main = "ARMA(1,2) ACF: Theoretical vs Sample",
             ylab = "ACF")


# AR(1) plots
plot_compare(ar_t_acvf, ar_s_acvf,
             main = "AR(1) ACVF: Theoretical vs Sample",
             ylab = "ACVF")


plot_compare(ar_t_acf, ar_s_acf,
             main = "AR(1) ACF: Theoretical vs Sample",
             ylab = "ACF")


# MA(2) plots
